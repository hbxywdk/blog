    • Vue.extend需要单独提一下

    1. 在Vue.extend的内部定义一个Sub，Sub继承于Super（指Vue）
    2. Vue.extend()返回Sub，Sub 是 Vue 的子类
    3. 所以 ‘new Vue({});’ 与 ‘var p = Vue.extend({}); new P();’ 实际上是类似的东西
